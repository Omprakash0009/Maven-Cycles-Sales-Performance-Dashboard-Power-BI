# Maven Cycles Business Intelligence Dashboard

Developed by **Om Prakash**

This repository showcases a comprehensive Power BI dashboard developed using the Maven Cycles dataset. The project transforms raw business data into meaningful insights through data modeling, DAX calculations, and interactive visualizations. It provides stakeholders with a clear understanding of sales performance, profitability, regional trends, and customer purchasing behavior.

---

# Project Goal

The Maven Cycles dataset contains business transactions across products, customers, and geographic regions. This dashboard was built to convert raw operational data into actionable business insights.

The dashboard enables users to:

- Monitor Revenue, Profit, and Profit Margin
- Analyze product performance across categories
- Compare regional sales performance
- Identify high-performing products
- Discover factors influencing product sales using AI visuals
- Explore revenue through interactive drill-down analysis
- Segment sales patterns using clustering techniques

The final solution provides an interactive and data-driven Business Intelligence experience for business users and decision-makers.

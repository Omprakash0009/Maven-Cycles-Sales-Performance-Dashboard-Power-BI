# Maven-Cycles-Sales-Performance-Dashboard-Power-BI
An interactive Power BI Sales Performance Dashboard for Maven Cycles featuring KPI reporting, revenue &amp; profit analysis, product performance, geographic insights, Key Influencers, Decomposition Tree, clustering analysis, and dynamic business intelligence dashboards using DAX and Power Query.

# Maven Cycles Sales Performance Dashboard

## Overview

This project presents an interactive Sales Performance Dashboard developed in Power BI using the Maven Cycles dataset.

The dashboard transforms raw transactional sales data into meaningful business insights through interactive visualizations, KPI tracking, and advanced analytical features.

---

## Business Objective

To analyze business performance by monitoring:

- Revenue
- Profit
- Profit Margin
- Product Performance
- Regional Sales
- Customer Segmentation
- Sales Trends

The dashboard enables business users to identify growth opportunities and make data-driven decisions.

### Key Influencers Analysis
